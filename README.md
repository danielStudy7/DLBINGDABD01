# DLBINGDABD01
Dieses Repository stellt den Prototypen aus SAS-Studio für den Kurs DLBINGDABD01 dar. 
## datenGenerator
Ein selbstgeschriebenes Python Programm, um die Quelldaten für den Prototypen erstellen zu können. Es werden jeweils Zufallswerte aus angegebenen Bereichen selektiert und Abweichungen bewusst integriert. Einzige Ausnahme sind die Abweichungen für den Output "testData.csv" diese wurden manuell hinzugefügt, nachdem die Daten erzeugt wurden.
